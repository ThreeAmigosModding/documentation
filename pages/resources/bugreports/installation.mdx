# Installation

This will go over the installation guide for TAM_BugReports. Please be sure to read this carefully and if you have any questions, feel free to ask in our Discord.

<Steps>

### Install & setup all dependencies
- [ox_lib](https://github.com/overextended/ox_lib/releases/latest)
- Make sure your server artifacts version is `9000` or above. Downloaded the [Windows](https://runtime.fivem.net/artifacts/fivem/build_server_windows/master/) or [Linux](https://runtime.fivem.net/artifacts/fivem/build_proot_linux/master/) artifacts.

### Download the latest release from [Keymaster](https://keymaster.fivem.net) or [Github](https://github.com/threeamigosmodding/TAM_BugReports/releases/latest) or build the source code.

### Add the resources to your server.cfg (or resources.cfg)

```bash
start ox_lib # Ensure this is started before it.
start TAM_BugReports
```
</Steps>
